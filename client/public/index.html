A ranking of the most frequently appearing stocks on investing.com's 52 week highs list.

  • Tech used: python (bs4 package for web scraping), MySQL, Express.js, Node.js, React.js
  • Ranking and market values update every day at 11:59 pm
  • Tap on column labels to sort items
  • Github for this project
  • API endpoint
